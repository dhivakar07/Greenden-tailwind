# 🌱 Greenden - Tailwind CSS Project

Greenden is a modern, responsive landing page built with **Tailwind CSS**.  
It demonstrates clean UI/UX, semantic markup, accessibility practices, and scalable workflow setup — ideal for portfolio presentation.

---

## 🚀 Features
- Fully responsive design (mobile, tablet, desktop)
- Built with **Tailwind CSS** utility-first classes
- Semantic HTML structure for accessibility
- Smooth transitions and hover effects
- Clean, minimal, recruiter-facing presentation

---

## 🛠️ Tech Stack
- **HTML5**
- **Tailwind CSS**
